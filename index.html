Leads Manager

A browser extension to manage projects leads of a feelancing team, Simple dashboard add edit delete or change the status of your lead with a single click. Uses REST API and also has user authentication.

About The Project

This is a browser extension made to manage the Upwork leads of a freelancer or a whole team. The extension is made specifically and works well on the chrome browser. The main pivot of a browser extension is a manifest.json file which specifies basic metadata about your extensions such as the name and version, and can also specify aspects of your extension's functionality such as background scripts, content scripts, and browser actions. The project includes various scripts mainly content.js which runs in the context of a particular web page, and can directly access and change the content of web pages and background.js which is a single script that manages the extension, these communicate with each other with Message API. Our extension also contains four HTML pages one for login, a dashboard, to add and editing leads which load when we click on the extension button on the top left of our browser. This project is made during my internship at Zasya Solutions, thanks to them for allowing the project to be public on GitHub. As I worked only on the frontend section of the extension it includes only that.
